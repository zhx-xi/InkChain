# InkOS 安全策略

> **⚠️ 法律免责声明**：本文档仅供参考之用，不构成法律建议。

**最后更新**：2026-07-01

## 概述

InkOS 将用户数据安全放在首位。作为一款本地优先的 AI 写作工具，我们致力于确保软件的安全性和透明度。

## 数据处理与存储

- **全部数据本地存储**：所有创作内容、配置、API 密钥均存储在您的本地设备上
- **无云端数据库**：InkOS 不连接任何集中式数据库或云端存储
- **无数据传输**：InkOS 不会将您的数据上传到任何外部服务器（除非您主动使用第三方 AI API）

## 报告安全漏洞

### 报告流程

1. **请勿公开披露**：请勿在 GitHub Issues、讨论区或其他公开渠道披露安全漏洞
2. **发送报告**：通过 GitHub Security Advisories 或提交 Issue 联系维护者

### 我们承诺

- **响应时间**：在 5 个工作日内确认收到您的报告
- **状态更新**：不超过每 14 天更新进展
- **修复时间表**：严重漏洞在确认后 30 天内发布修复

## 版本支持

| 版本 | 安全更新支持 |
|------|-------------|
| v1.5.x（最新） | ✅ 支持 |
| v1.4.x 及更早 | ❌ 不再支持 |
